CSS

A CSS-t html-el együtt kezdtük el tanulni, szintén az alapokkal kezdve

A tanár úr, html-hez hasonlóan, sőt, ugyanazokban a feladatokban segített megjegyezni CSS parancsokat, mint amiben a html-t tanultuk.

Ez a felsorolás pár példát ad CSS parancsokra:

  1. color: A szöveg színének beállítása. Példa: color: blue;
  2. background-color: A háttérszín beállítása. Példa: background-color: #f2f2f2;
  3. font-size: A betűméret beállítása. Példa: font-size: 18px;
  4. font-family: A betűtípus beállítása. Példa: font-family: Arial, sans-serif;
  5. text-align: A szöveg igazítása (balra, jobbra, középre). Példa: text-align: center;

A css mellett tanulhattak a kicsit jobban érdeklődők Tailwind CSS-t, és daisyUI-t.

A Tailwind egy CSS keretrendszer, amivel hatékonyabban tudunk kódot írni. Egy sorozat előre megadott osztályt biztosít, amelyekkel gyorsan és könnyen tudunk formázni.

A daisyUI pedig egy Tailwinden alapuló keretrendszer, ami további osztályokat biztosít, ezzel méginkább megkönnyítve a dolgunkat.

Ez pedig egy példa a Tailwind CSS használatára:

Sima CSS:

                            body {
                                font-family: Arial, sans-serif;
                                background-color: #f2f2f2;
                              }
                              h1 {
                                text-align: center;
                                color: blue;
                                font-size: 36px;
                              }
                              p {
                                font-size: 18px;
                                text-align: justify;
                              }

Tailwind CSS:

                            body class="bg-gray-200 font-sans"
                            h1 class="text-center text-blue-500 text-4xl">Teszt oldal h1
                            p class="text-lg text-justify">Ez egy teszt oldal, amelyen néhány alapvető CSS parancs látható.
